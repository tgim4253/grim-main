Bundled UI fonts used by Grim's design tokens.

- Inter Regular, Medium, SemiBold, and Bold were downloaded from Google Fonts.
- Liberation Mono Regular and Bold were downloaded from the Liberation Fonts 2.1.5 release archive.

Both font families are distributed under the SIL Open Font License.
